# v5.5 唐宋主界面术语桥与地图图层读法记录

本文记录 v5.5 的首轮小切片：先把默认唐宋主界面里的 HUD、地图图层、信息面板、军令按钮和战报分类从二战/英文调试口径桥接到唐宋读法。它不是 v5.5 发布级 UI 完整收口；本轮没有重绘地图美术、棋子符号、粮道线、计划箭头，也没有做运行时截图或布局验收。

## 1. 目标

- 让默认 `jianlong_960_unification` 场景第一屏少显示玩家可见英文调试词。
- 保持 legacy 阿登路径原显示不变。
- 给 `MapDisplayLayer`、`GamePhase`、HUD、Command 面板和 EventLog 提供 `isTangSongScenario` 显示桥。
- 只改变显示文案和只读分类，不改变 command、rule、log、Codable raw case、存档或 AI 执行权限。

## 2. 非目标

- 不重命名 `Faction`、`Division`、`ComponentType`、`ProductionKind`、`GamePhase`、`MapDisplayLayer` raw case。
- 不改变 `Command` / `ZoneDirective -> WarCommandExecutor -> RuleEngine` 统一执行管线。
- 不新增规则、AI 策略、外交归附、天命治理或胜利结算。
- 不修改 SpriteKit 地图美术、棋子 icon、地形 palette 或布局架构。
- 不运行本机 Xcode build/test、模拟器、Probe、Smoke、Stage Regression、Dynamic Theater Regression、Full 或性能测试。
