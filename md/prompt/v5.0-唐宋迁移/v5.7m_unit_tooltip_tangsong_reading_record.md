# v5.7m 常驻军队 tooltip 唐宋读法记录

本文记录唐宋 v5.7m 小切片：补齐右下角常驻选中军队 tooltip 的唐宋读法。该切片只做玩家可见显示桥，不改变单位模型、补给规则、退却规则、命令管线或任何战术权威。

## 目标

- 让 `UnitTooltipView` 在唐宋场景下显示兵种、兵力、补给、退却和本回合。
- 将 tooltip 中的 ART/ARM/MOT/INF、Supplied/Low/Encircled、Retreatable/Hold、Yes/No 显示为器械/禁军/骑军/厢军、有粮/缺粮/被围、可退/固守、已行动/未行动。
- 让 tooltip 的 accessibility label 同步使用唐宋读法，避免读屏仍朗读英文调试口径。
- 保持非唐宋 legacy 路径原显示不变。

## 非目标

- 不改 `Division`、`ComponentType`、`SupplyState`、`RetreatMode` 或存档 schema。
- 不改变补给、退却、行动状态、战斗、围城、胜负或 AI 决策规则。
- 不新增命令，不调用 `RuleEngine`，不写 `GameState` 或 `eventLog`。
- 不做截图、iPhone/iPad 横竖屏布局验收、VoiceOver 实机验收或本地 build/test。
- 不处理本轮子 Agent 扫出的全部 v5.8 RC 残留；AI 面板默认主路径残留已由 v5.8a 首轮处理，diagnostics、错误原文、raw JSON 调试区和其他面板残留仍留给后续 RC 审计或独立显示桥切片。

## 实现说明

```text
RootGameView
  -> UnitTooltipView(
       division: selectedDivision,
       isTangSongScenario: gameState.isTangSongScenario
     )

UnitTooltipView
  -> legacy: Type / Strength / Supply / Retreat / Acted
  -> 唐宋: 兵种 / 兵力 / 补给 / 退却 / 本回合
```

唐宋显示桥：

- `Division.tooltipTypeCode(isTangSongScenario:)`
  - artillery -> 器械
  - armor -> 禁军
  - motorizedInfantry 权重 >= 0.40 -> 骑军
  - fallback -> 厢军
- `SupplyState.tooltipDisplayName(isTangSongScenario:)`
  - supplied -> 有粮
  - lowSupply -> 缺粮
  - encircled -> 被围
- `RetreatMode.tooltipDisplayName(isTangSongScenario:)`
  - retreatable -> 可退
  - hold -> 固守
- `actedText`
  - true -> 已行动
  - false -> 未行动
