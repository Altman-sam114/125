# v5.8t accessibility / VoiceOver 可读文案硬化记录

完成日期：2026-07-07

## 目标

在不改变规则、布局、focus order、hit target、SpriteKit 地图交互或 Codable schema 的前提下，继续收口唐宋默认路径中读屏会朗读出的固定英文、占位词、状态不明按钮和 MapEditor 可访问名称不足问题。

## 实现摘要

- `RootGameView` 信息面板按钮增加 accessibility label/value/hint，读屏可知道面板当前已展开或已收起。
- 主棋盘 accessibility value 从当前 `selectedHex` / `selectedRegionId` 只读派生选中地块与州府名称；未选中时朗读“尚未选中地块”。
- `InfoPanelToggle` 通用“详情”按钮增加展开/收起状态和提示。
- `GeneralProfileView` 将领头像占位改为装饰性元素，不再朗读“头像占位 / portrait placeholder”。
- MapEditor 底图偏移输入框增加“底图横向偏移 / 底图纵向偏移”可访问标签。
- MapEditor 地图编辑画布增加中文 accessibility label/hint。

## 架构边界

- Hex 仍是战术权威；Region / Theater / Front / Deploy 边界不变。
- 玩家、AI、聊天命令和 MockAI 仍必须通过 `Command` / `ZoneDirective -> WarCommandExecutor -> CommandValidator -> RuleEngine` 执行。
- 本轮不改变 `GameState`、命令、规则、AI 决策、导出 JSON schema、raw enum、legacy `.allies/.germany` 桥或任何 Codable raw 值。
- 本轮不是完整 VoiceOver 实机验收，不实现逐 hex 可聚焦地图操作路径。
